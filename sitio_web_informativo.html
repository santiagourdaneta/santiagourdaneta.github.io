Sitio Web Informativo

Creo el sitio web informativo para que des a conocer información de un negocio en Internet. Es como el folleto o tríptico de tu negocio

Podras listar tus productos a modo de catálogo con la posibilidad de recibir pedidos.

Algunas caracteristicas:

  1. Sencillas. Las páginas web informativas generalmente son sencillas. Como su objetivo es sólo informar, se componen principalmente de textos, imágenes, y en ocasiones, video insertado.
  2. Estáticas. El contenido de una página informativa, aparece de manera estática en el sitio para ser leída por el visitante.
  3. Un sitio web informativo tiene la finalidad de sólo informar y nada más. No son como las landing pages, que permite interactuar de diferentes maneras con el usuario.
